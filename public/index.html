General Assembly Web Development Immersive Coursework

I recently graduated from Genral Assembly's Web Development Immersive Program. Below are the projects I worked on during the course.

Venu Menu:

VenuMenu was our group project completed at the halfway point of the program. The menu of venues is pulled from the Songkick API. The app allows a user to search venues by name or location. A search yields venu results. Each venue result includes the name, address, telephone number, and a link to the venue website. My focus on this project was the mainly front end development. It was the first time I was able to translate a photoshop file into an interactive website accurately. This was exciting to me because it marked the moment I realized I had truly discovered a new creative platform. Check out the github repository or see the website in action here.
